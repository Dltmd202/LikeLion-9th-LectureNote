### 1.4.1 HTML 코드

***

* 이거 html문서야~~~

  >```html
  ><html>
  >  
  ></html>
  >```

* 문서 정보에 대한 태그

  > ```html
  > <head>
  >   
  > </head>
  > ```

* 직접적으로 나타나는 태그

> ```html
> <body>
>   
> </body>
> ```

### 1.4.2 예제

***

```html
<!DOCKTYPE html>
<html lang="ko">
 <head>
  	<meta charset = "utf-8">
  	<title>나를 소개해요</title>
</head>
<body>
  <h1>이승환을 소개합니다.</h1>
  <h2>이승환을 소개합니다.</h2>
</body>
</html>
```
