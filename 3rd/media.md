### 3.4.1 이미지 태그

```html
<img src="https://dltmd202.github.io/res/IMG_0328.JPG" width= "200px" alt="이미지를 불러올 수 없습니다.">
```

<img src="https://dltmd202.github.io/res/IMG_0328.JPG" width= "200px">

> **src**
>
> >  불러올 이미지의 URL 속성 값으로 가짐 
>
> **alt**
>
> > 불러올 이미지가 없거나 불러오는데 실패했을 경우 대신 표시되는
> >
> > 문장
>
> weight / height
>
> > 이미지의 높이와 너비 설정
> >
> > * **CSS**에서 조정 권장
