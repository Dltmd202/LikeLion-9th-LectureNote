### 3.1.2 활용 방법

```html
<!DOCTYPE html>
<html land="kr">
  <head>
    <meta charset="utf-8">
    <title>승환을 소개합니다.</title>
  </head>
  <h1>안녕하세요 저는 승환입니다.</h1>
  <p></p>
</html>
```



* ```html
  <!DOCTPYE html>
  ```

> 문서 형식을 정의



* ``` html
  <html land="kr">
  ```

> 본격적인 태그의  시작 사용하는 주 언어 정의



* ``` html
  <meta charset="utf-8">
  ```
